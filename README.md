# ClubOS

Sistema de gestão de clube de futebol — elenco, táticas, partidas, estatísticas e carreira.

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```
