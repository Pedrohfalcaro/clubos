# FIFA Career Companion

## Objetivo

Criar uma plataforma complementar ao Modo Carreira do EA FC/FIFA.

O usuário joga as partidas normalmente no jogo e utiliza este sistema para gerenciar:

- Elenco
- Finanças
- Contratações
- Diretoria
- Categorias de Base
- Históricos
- Estatísticas
- Eventos Aleatórios
- Narrativas Dinâmicas

---

# Filosofia

O FIFA gera o resultado da partida.

O sistema gera as consequências.

Exemplos:

- Craque lesionado fora por 4 meses
- Patrocinador oferece bônus
- Investidor injeta dinheiro
- Torcida protesta
- Jogador pede aumento
- Jogador quer sair
- Escândalo extracampo

---

# Sistemas

## Elenco

Cada jogador possui:

- Nome
- Idade
- Número
- Posição
- Salário
- Valor de Mercado
- Moral
- Potencial
- Status

Status:

- Titular
- Reserva
- Promessa
- Transferível
- Emprestado

---

## Finanças

Receitas:

- Premiações
- Bilheteria
- Patrocínio
- Venda de Jogadores

Despesas:

- Salários
- Contratações
- Rescisões
- Estrutura

---

## Diretoria

Indicadores:

- Confiança da Diretoria
- Confiança da Torcida
- Saúde Financeira
- Desempenho Esportivo

Escala:

0-100

---

## Eventos Aleatórios

Os eventos são disparados após partidas.

Categorias:

- Positivos
- Negativos
- Neutros
- Históricos

---

## Estatísticas

Registrar:

- Jogos
- Vitórias
- Empates
- Derrotas
- Gols Pró
- Gols Contra

Jogadores:

- Jogos
- Gols
- Assistências
- Cartões
